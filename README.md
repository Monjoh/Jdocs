# jDocs

A lightweight cross-platform desktop app for organizing files with tagging, categorization, metadata extraction, and search.

## Features (Planned)
- Drag & drop file input with metadata extraction
- Auto-suggest project/folder placement
- Tag, categorize, and comment on files
- Search across filenames, metadata, tags, and categories
- Supports: Excel, PowerPoint, Word, images, code files
- Fully local â€” no cloud, no external APIs

## Platform Support
- Windows (user-level install, no admin rights needed)
- macOS

## Requirements
- Python 3.10+

## Setup
```bash
pip install -r requirements.txt
python src/main.py
```
